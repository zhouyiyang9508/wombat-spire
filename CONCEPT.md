# 🗡️ Wombat Spire — 修仙塔

> 一款中国修仙背景的 Roguelike 卡牌游戏，运行于浏览器，灵感来自《杀戮尖塔》

---

## 核心概念

你是一名散修，独自攀登**通天灵塔**。塔内险象环生——妖兽横行，魔修肆虐，机缘与危机并存。
每一次登塔，地图随机生成，卡牌随缘获取，境界逐步提升。能否登顶，全凭道心与机运。

---

## 核心玩法（类杀戮尖塔）

### 地图
- 分层随机地图（15-20 层）
- 节点类型：普通战斗 / 精英战斗 / 事件 / 休息地 / 商店 / Boss
- 每层有多条路可选，策略选路是核心

### 战斗系统
- **回合制卡牌战斗**
- 每回合抽 N 张牌，消耗灵气打出
- 卡牌类型：攻击、防御、法术、辅助
- 敌人有意图（提前显示下一步行动）

### 修仙成长体系

| 境界 | 说明 |
|------|------|
| 炼气期 | 游戏开始，基础卡组 |
| 筑基期 | 第一个 Boss 后突破，解锁新卡牌类别 |
| 金丹期 | 第二个 Boss 后，战斗机制变化 |
| 元婴期 | 最终 Boss 前，强力技能解锁 |

### 卡牌体系
- **道法**（攻击）：剑气纵横、雷法轰鸣、火球术
- **身法**（防御/闪避）：御风术、铁布衫、缩地成寸
- **灵符**（法术/特效）：封印符、爆炎符、聚灵符
- **器使**（法宝装备）：飞剑、护山铃、乾坤袋
- **丹药**（消耗/增益）：回灵丹、破障丹、化毒丹

### 遗物系统（类杀戮尖塔遗物）
- 被动效果永久生效
- 主题：法宝碎片、宗门令牌、上古秘典

### 事件系统
- 随机遭遇：问道老者、神秘洞府、劫难
- 选择分支影响资源/卡牌/状态

## 技术方案

- **纯前端 HTML/CSS/JavaScript**，无需后端
- 单文件 `index.html` 可直接打开，也可部署 GitHub Pages
- 可能引入轻量框架：Phaser.js（游戏框架）或 纯 vanilla JS
- 本地 `localStorage` 保存游戏进度（单存档）
